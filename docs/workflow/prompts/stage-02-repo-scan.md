# Trigger: Stage 02 - Repo Reconnaissance

## Read First
- .grok/repository-map.md (once created)
- Full current repo tree
- Key existing files (README, main scripts, configs)

## Task
Produce an accurate map of what exists, entrypoints, conventions, and gaps. Base it only on actual inspection.

## Focus
Infrastructure nature of the repo (Bash orchestration, Nginx, monitoring configs).

## Constraints
- No hallucination of non-existent code.
- Identify where new provider support (AIOZ, Dynex) would plug in later.

## Output
Complete docs/workflow/active/02-repo-scan.md using the standard structure.

## Stop Condition
Scan documented. Move to architecture stage.
